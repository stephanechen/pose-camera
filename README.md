# 透视取景器 - AI拍照参照小程序

> 跟小红书学拍照，透明参考图叠加取景

## 📱 产品简介

一款 AI 智能取景参照拍照工具，帮助用户精准复刻小红书等平台的网红照片。用户上传参考图后，透明叠加到相机画面，辅助构图取景。

## ✨ 核心功能

- **参考图上传** - 从相册选择参考图
- **透明叠加** - 参考图以半透明方式叠加到相机画面
- **手势交互** - 单指拖拽、双指缩放/旋转参考图
- **苹果风格倍数切换** - 0.8x / 1x / 2x / 5x 焦距快速切换
- **九宫格辅助线** - 三分法构图辅助
- **静默登录** - 自动获取用户身份，无需弹窗授权

## 🛠 技术栈

- **前端**: 微信小程序原生开发（WXML / JS / WXSS）
- **相机**: 小程序 Camera 组件 + CameraContext API
- **渲染**: Canvas 2D（多层叠加）
- **登录**: 静默登录（wx.login 获取 openid）

## 🚀 快速开始

### 1. 导入项目
使用**微信开发者工具**导入项目目录。

### 2. 修改 AppID
在 `project.config.json` 中将 `appid` 改为你的小程序 AppID（需在 [微信公众平台](https://mp.weixin.qq.com/) 注册）。

### 3. 配置权限
在微信公众平台的「开发管理」→「开发设置」中，配置以下权限：
- 相机（camera）
- 相册（photos）

### 4. 体验
直接在模拟器或真机上打开小程序即可使用。
