# TabBar 图标说明

本目录放置 TabBar 图标文件。

## 需要的图标

请在微信开发者工具中右键 assets/icons 目录，选择「新建图片」：

| 图标名称 | 尺寸 | 说明 |
|---------|------|------|
| home.png | 81×81 px | 首页 - 未选中状态 |
| home-active.png | 81×81 px | 首页 - 选中状态 |
| camera.png | 81×81 px | 取景 - 未选中状态 |
| camera-active.png | 81×81 px | 取景 - 选中状态 |
| user.png | 81×81 px | 我的 - 未选中状态 |
| user-active.png | 81×81 px | 我的 - 选中状态 |

## 临时解决方案

如果图标尚未准备好，可以：
1. 使用 emoji 作为临时图标（修改 app.json 中的 iconPath）
2. 或从微信开放文档下载示例图标

## 图标设计规范

- 格式：PNG（支持透明）
- 尺寸：81×81 像素
- 设计风格：线性图标，圆角
- 颜色：与选中态主色 #FF6B6B 搭配
